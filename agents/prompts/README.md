# Agent Prompts

This directory contains prompts for sub-agents that research and populate data.

## Usage

Copy these prompts into your preferred AI assistant (Claude, Gemini, GPT-4, etc.) to generate data that matches our YAML schemas.

## Available Prompts

- `job_research.md` - Research job AI exposure data
- `location_research.md` - Research country AI adoption rates
- `quote_research.md` - Find expert quotes on AI displacement
- `quip_generation.md` - Generate dark humor one-liners

## Schemas

See `schemas/` for YAML schema definitions that generated data must match.
